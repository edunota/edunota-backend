# edunota backend

## Dev enviroment
We use [Air](https://github.com/cosmtrek/air) on dev enviroment for hot reloads

### Spin up dev enviroment
```bash
# this will spins up docker-compose file
make spinup
```
## start server on dev mode
```bash
make dev
```

## Tests

running unit tests
```bash
make test
```

runing e2e tests
```bash
make test-e2e
```

runing code coverage test
```bash 
make test-cov
```
